Spurious Correlations: correlation is not causation — thousands of charts of real data showing actual correlations between ridiculous variables.

Examples from the project: Bachelor's degrees awarded in Psychology correlates with the number of groundskeepers in Utah. Popularity of the first name Christopher correlates with burglaries in Oklahoma. Sales of LP/Vinyl Albums correlates with Costco Wholesale's stock price. Per capita consumption of margarine correlates with the divorce rate in Maine. The number of judges in Indiana correlates with viewership of The Big Bang Theory.

Why this works:

Data dredging: the project compares over 25,000 variables in its database against each other to find ones that randomly match up — over 636 million correlation calculations. Instead of starting with a hypothesis and testing it, the project tosses a bunch of data in a blender to see what correlations shake out. This is dangerous as a general analysis method, because any sufficiently large dataset will yield strong correlations completely at random.

Lack of causal connection: there is probably no direct connection between most of the variable pairs shown, despite how convincing the chart looks.

Observations not independent: for many variables, sequential years are not independent of each other — you will often see trend-lines form. If a population is continuously doing something every day, there's no reason to think it would suddenly change on January 1st. A naive p-value calculation does not account for this, which produces a lower "chance of randomness" than actually represents reality.

Y-axes doesn't start at zero: the graphs' y-axes are often truncated, and line graphs make the visual connection stand out more than it deserves — visually deceptive because the only real data is at the points on the graph, not the lines connecting them.

Confounding variable: a "sneaky third" variable (like a shared trend, or in some cases a global event) can influence both plotted variables independently, making them look connected when they aren't.

Outliers: a single outlier in a small dataset can substantially increase an apparent correlation.

Low n: some of the charted comparisons don't have many data points — even if the p-value looks acceptable, few datapoints combined with data dredging should raise suspicion.
